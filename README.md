# Perspectives

Perspectives is an open-source database client that turns rigid table-browsing into reusable, shareable, AI-steerable *perspectives* — saved presentations of data with their own columns, filters, sorts, joins, and (in shared mode) permissions. It runs as an Electron desktop app, a self-hostable server, or a managed SaaS, with the same engine under the hood and PostgreSQL as the v1 target database.

See [`perspectives-plan.md`](./perspectives-plan.md) for the full product and build plan.
